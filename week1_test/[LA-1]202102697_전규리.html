교과목 개요

수업 개요

본 수업에서는 클라언트 웹 기술인 HTML5, CSS3, JavaScript와 서버 기술인 PHP등 웹 프로그래밍과 관련된 기술을 이해한다.그리고 실습을 통해 웹 사이트 또는 웹 앱을 구축해 봄으로써 최신 웹 표준기술을 습득한다.

수업 운영 방식

  1. 대면 수업으로 함
  2. 학교 사이버 캠퍼스를 통한 모든 강의 동영상, 슬라이드, 공지 사항 제공
  3. Flipped Learing 방식
    • 이론 수업 전 예습 : 이론 강의 동영상을 통해 예습
    • 이론 수업 : 그룹 중심의 토론 및 문제 해결
    • 실습 수업 : 개별적으로 실습 과제를 해결
